# Health Tracker — เว็บแอปดูแลสุขภาพ

เว็บแอปสำหรับบันทึกและติดตามสุขภาพ ทำงานในเบราว์เซอร์ พร้อมเชื่อมต่อ Google Sheets เพื่อเก็บข้อมูลถาวร

## ฟีเจอร์

- **ภาพรวม (Dashboard)** — สรุปแคลอรี่ น้ำหนัก BMI น้ำดื่ม การนอน พร้อมกราฟ 7 วัน / 30 วัน
- **บันทึกอาหาร** — แยกตามมื้อ พร้อมแคลอรี่
- **ออกกำลังกาย** — บันทึกประเภท ระยะเวลา และแคลอรี่ที่เผาผลาญ
- **น้ำหนัก & BMI** — กราฟติดตามระยะยาว คำนวณ BMI อัตโนมัติ แบ่งหมวด (ผอม/ปกติ/น้ำหนักเกิน/อ้วน)
- **น้ำดื่ม & การนอน** — ปุ่มลัด +200/+350/+500/+750 ml และบันทึกคุณภาพการนอน
- **เป้าหมายรายวัน** — กำหนดเป้าหมายแคลอรี่และน้ำดื่มได้เอง
- **ซิงค์ Google Sheets** — เก็บข้อมูลทั้งหมดในชีตของคุณ (เป็นเจ้าของข้อมูลเอง)
- **Export / Import** — สำรองข้อมูลเป็นไฟล์ JSON

## โครงสร้างไฟล์

```
health-app/
├── index.html       ← หน้าเว็บหลัก (เปิดไฟล์นี้)
├── styles.css       ← ดีไซน์
├── app.js           ← Logic ของแอป (UI, ฟอร์ม, กราฟ)
├── storage.js       ← จัดการ LocalStorage และเรียก Google Sheets
├── Code.gs          ← โค้ด Google Apps Script (วางในเว็บ Apps Script)
└── README.md        ← ไฟล์นี้
```

## วิธีใช้แบบเร็ว (Local Mode)

1. ดับเบิลคลิก `index.html` เพื่อเปิดในเบราว์เซอร์
2. เริ่มบันทึกข้อมูลได้เลย — ข้อมูลจะเก็บใน LocalStorage ของเบราว์เซอร์
3. หากต้องการเก็บข้อมูลถาวรและเปิดดูจากเครื่องอื่นได้ ทำตามขั้นตอนเชื่อมต่อ Google Sheets ด้านล่าง

> หมายเหตุ: ในโหมด Local Mode ข้อมูลจะอยู่เฉพาะในเบราว์เซอร์เครื่องนั้น และจะหายถ้าล้าง browser data

## เชื่อมต่อ Google Sheets

### ขั้นตอนที่ 1 — สร้าง Google Apps Script

1. ไปที่ <https://script.google.com> แล้วกด **New project**
2. เปลี่ยนชื่อโปรเจกต์เป็น `Health Tracker API` (หรือชื่ออะไรก็ได้)
3. ลบโค้ดเริ่มต้นในไฟล์ `Code.gs` ออกทั้งหมด
4. คัดลอกเนื้อหาในไฟล์ `Code.gs` (ในโฟลเดอร์นี้) ไปวาง
5. **แก้ค่า `SECRET`** ที่ด้านบนของโค้ดเป็นรหัสลับของคุณเอง เช่น `'my-secret-2026'`
   - รหัสนี้ใช้ป้องกันคนอื่นเรียก API ของคุณ
   - จดไว้ เดี๋ยวต้องใส่ในเว็บแอปด้วย
6. กด **Save** (Ctrl/Cmd + S)

### ขั้นตอนที่ 2 — Deploy เป็น Web App

1. กดปุ่ม **Deploy** มุมขวาบน → **New deployment**
2. กดไอคอนเฟือง ⚙ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่า:
   - **Description:** `Health Tracker` (หรืออะไรก็ได้)
   - **Execute as:** `Me` (อีเมลของคุณ)
   - **Who has access:** `Anyone`
4. กด **Deploy**
5. ครั้งแรกจะมีหน้าต่างให้อนุญาตสิทธิ์ — กด **Authorize access** → เลือกบัญชี Google ของคุณ
   - ถ้าเจอหน้า "Google hasn't verified this app" ให้กด **Advanced** → **Go to (project name) (unsafe)** → **Allow**
   - (ปลอดภัย เพราะมันคือสคริปต์ของคุณเอง)
6. คัดลอก **Web app URL** ที่ปรากฏ — มีหน้าตาแบบนี้:
   ```
   https://script.google.com/macros/s/AKfycby.../exec
   ```

### ขั้นตอนที่ 3 — ตั้งค่าในเว็บแอป

1. เปิด `index.html` → ไปที่หน้า **ตั้งค่า**
2. วาง **Web app URL** ในช่อง "Google Apps Script Web App URL"
3. ใส่ **SECRET** เดียวกับที่ตั้งในโค้ด `Code.gs`
4. กด **บันทึก** แล้วกด **ทดสอบการเชื่อมต่อ**
5. ถ้าขึ้น "✓ เชื่อมต่อสำเร็จ" — ใช้งานได้แล้ว
6. หลังจากนี้ ทุกครั้งที่บันทึกหรือลบข้อมูล จะส่งไป Google Sheet โดยอัตโนมัติ

> **Tip:** Apps Script จะสร้างไฟล์ Google Sheet ใหม่ชื่อ "Health Tracker Data" ในไดรฟ์ของคุณ ลองเข้าไปดูได้ที่ <https://drive.google.com>

### ปุ่มเพิ่มเติมในหน้าตั้งค่า

- **ส่งข้อมูลทั้งหมดไป Sheet** — อัปโหลดข้อมูลที่อยู่ในเบราว์เซอร์ทั้งหมดไป Sheet (เขียนทับของเดิม)
- **ดึงข้อมูลจาก Sheet** — โหลดข้อมูลจาก Sheet มาแทนที่ของในเบราว์เซอร์
  - มีประโยชน์เมื่อเปิดเว็บแอปจากเครื่องอื่น
- **ส่งออก JSON / นำเข้า JSON** — สำรองข้อมูลแบบ offline

## การคำนวณ BMI

แอปใช้สูตรมาตรฐาน: `BMI = น้ำหนัก (kg) ÷ (ส่วนสูง (m))²`

หมวดหมู่ (ตามเกณฑ์เอเชีย):
- น้อยกว่า 18.5 = ผอม
- 18.5 – 22.9 = ปกติ
- 23.0 – 24.9 = น้ำหนักเกิน
- 25.0 ขึ้นไป = อ้วน

## คำถามที่พบบ่อย

**Q: ปลอดภัยไหม? SECRET เก็บอยู่ที่ไหน?**
A: SECRET เก็บใน LocalStorage ของเบราว์เซอร์เท่านั้น ไม่ได้ส่งไปที่ไหน นอกจาก Apps Script ของคุณเอง อย่างไรก็ตาม ไม่ควรเปิดเว็บแอปนี้บนเครื่องที่คนอื่นใช้ร่วมกัน

**Q: เปิดจากมือถือได้ไหม?**
A: ได้ครับ ดีไซน์รองรับหน้าจอเล็ก หรือจะอัปโหลดทั้งโฟลเดอร์ไปโฮสฟรี เช่น GitHub Pages, Netlify ก็ได้

**Q: ข้อมูลของผมส่งไปไหนบ้าง?**
A: ส่งไปที่ Google Apps Script ของคุณเอง → เขียนลง Google Sheet ของคุณเอง ไม่มี server ตัวกลาง

**Q: อยากเปลี่ยนสี / ฟอนต์ ทำยังไง?**
A: แก้ตัวแปร CSS ในส่วนบนสุดของ `styles.css` (`:root { --primary: ... }`)
